Info

Trashcan is a SPA for mapping trash on a map. This app is in theme of the 'Gentse Feesten'.

Users can report trash they encounter. Take a picture, define the size and add a type.

Technology:

Trashcan uses AngularJS for the frontend. In the back Slimframework is running in harmony with Redbean.
